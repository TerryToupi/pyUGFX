Detailed Description

Enumerations for input and graphics systems.

This file defines strongly-typed enumerations used for keyboard and mouse input (namespace input) and for graphics API abstractions (namespace gfx), such as shader stages, buffer types, texture formats, and rendering states. It also provides bitwise operator overloads for select enums to facilitate flag/mask usage.
