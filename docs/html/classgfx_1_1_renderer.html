Detailed Description

Main interface for the rendering system.

The Renderer is responsible for:

  • Initializing the rendering backend and GPU resources.
  • Shutting down and cleaning up resources.
  • Creating command buffers for recording GPU work.

This is typically implemented by a specific rendering backend (e.g., Vulkan, Metal) and accessed via the static instance.
