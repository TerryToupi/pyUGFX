Detailed Description

Defines data structures and utilities for encoding and decoding GPU draw/dispatch commands.

This file contains definitions for Draw and Dispatch command structures, along with helper classes to encode and decode these commands into a compact stream format.

Bitmask macros are also provided to track "dirty" state changes in command streams.
