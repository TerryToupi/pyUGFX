Detailed Description

template<typename U, typename V>
class utils::Pool< U, V >

Generic pool for managing objects and tracking their lifetimes using handles.

The Pool class enables fast allocation and deallocation of objects of type U. Each object is identified by a handle of type V, which includes an index and generation for safe access. Handles become invalid when their generation is incremented (i.e., after object removal).

Template Parameters
UType of objects stored in the pool.
VType used for handle generation.
