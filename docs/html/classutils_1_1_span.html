Detailed Description

template<typename T>
class utils::Span< T >

Non-owning view over a contiguous sequence of objects.

The Span class provides a convenient, bounds-checked interface for accessing arrays, initializer lists, or raw memory regions as a sequence, without managing their lifetime. It is especially useful for function arguments, simplifying code that works with slices of data.

Template Parameters
TType of elements in the sequence.
