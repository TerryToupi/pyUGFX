Detailed Description

template<typename T>
class utils::Handle< T >

Generic handle for resource identification and validation.

The Handle class provides a unique identifier composed of an index and a generation. Handles can be checked for validity, compared, packed/unpacked to/from an integer, and used with resource pools.

Template Parameters
TThe type of resource the handle refers to.
